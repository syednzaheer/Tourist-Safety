# Tourist Website

A comprehensive tourist information website designed to assist travelers with navigation, safety, and local insights.

## Features
- **AI Assistant**: Integrated AI support for tourist queries (`AI.html`).
- **Safety First**: SOS features (`sos.html`) and police database integration (`dbpolice.html`).
- **Navigation**: Map integration (`map.html`).
- **Local Guide**: Information on languages (`languages.html`) and tips (`tips.html`).
- **IoT Integration**: Smart features for tourists (`iot.html`).

## Usage
Open `index.html` in your web browser to start exploring.
